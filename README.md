# 🧮 Advanced BMI Calculator

A smart Python BMI Calculator that calculates Body Mass Index using different height units like meters, centimeters, and inches. The program also provides BMI classification and basic health advice.

---

## ✨ Features

- ✅ BMI Calculation
- ✅ Multiple Height Units Support
- ✅ Input Validation
- ✅ Health Category Detection
- ✅ Health Advice
- ✅ User-Friendly Command Line Interface
- ✅ Repeat Calculation Option

---

## 📌 BMI Formula

\[
BMI = \frac{weight(kg)}{height(m)^2}
\]

## 📊 BMI Categories

| BMI Range | Category |
|-----------|-----------|
| Below 18.5 | Underweight |
| 18.5 - 24.9 | Normal Weight |
| 25 - 29.9 | Overweight |
| 30 - 34.9 | Obese |
| 35+ | Severely Obese |
